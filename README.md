# 悠然工具箱

一款轻量级的桌面工具箱应用，专注于提升日常办公效率。功能持续增加中...

## 功能特性

### 批量重命名

批量重命名文件，支持多种命名规则组合。

- **命名规则**：
  - 序号：自定义起始值和位数
  - 文本：插入自定义文本
  - 当前时间：多种日期时间格式
  - 创建时间/修改时间：使用文件的时间属性
  - 文件大小：自动或指定单位
  - 原名称：完整保留、替换、截取
  - 连接符：常用分隔符
  - 随机字符串：可配置长度和字符类型
- **拖拽排序**：自由调整规则顺序
- **文件排序**：按名称、大小、创建时间、修改时间排序
- **复制模式**：
  - 原地复制：复制到原文件相同目录
  - 归档复制：复制到指定目录，支持新建子文件夹
- **手动编辑**：点击预览名称可单独修改

### 分类归档

将 Excel 数据按指定列自动分类到不同的 Sheet 或独立文件中。

- 支持同时选择多个 Excel 文件（.xlsx、.xls 格式）
- 可选择任意列作为分类依据
- 支持多行表头（1-6 行可选）
- 统一设置模式：所有文件使用相同的分类列和表头行数
- 单独设置模式：每个文件独立配置
- 分类方式：新建 Sheet 或输出为独立文件
- 自动保留原始数据在「全部」Sheet 中
- 保留原始表格的合并单元格和列宽设置
- 可自定义文件名前缀和后缀
- 处理完成后显示分类统计信息

### 图片压缩/转换

批量压缩图片或转换格式。

- 支持 JPEG、PNG、WebP 格式
- 可调节压缩质量（1%-100%）
- 支持设置最大宽度，超出时等比缩小
- 统一设置或单独设置每张图片的参数
- 复制模式：
  - 原地复制：保存到原图片相同目录
  - 归档复制：保存到指定目录

### PDF 工具

PDF 合并与拆分。

- **合并 PDF**：
  - 支持选择多个 PDF 文件
  - 拖拽调整合并顺序
  - 显示每个文件的页数
- **拆分 PDF**：
  - 每页一个文件
  - 按范围拆分（如：1-3, 5, 7-10）

### 二维码

生成和识别二维码。

- **生成二维码**：
  - 支持文本、网址等内容
  - 可调节尺寸、容错级别、边距
  - 多种配色方案，支持自定义颜色
  - 码点形状：方形、圆角、超圆角、经典
  - 定位点形状：方正、圆角、菱形、星形等 12 种样式
  - 支持添加 Logo，可调节大小、圆角、边距
  - 输出格式：PNG、JPEG、WebP
  - 支持下载和复制到剪贴板
  - 测试识别功能验证二维码可用性
- **识别二维码**：
  - 支持选择图片或粘贴剪贴板图片
  - 识别结果可一键复制

### 文本对比

对比两段文本的差异。

- 三种对比模式：按字符、按单词、按行
- 高亮显示新增、删除、未变部分
- 支持交换两侧文本
- 对比结果可复制

## 多语言支持

支持 10 种语言：

- 简体中文
- 繁体中文
- English
- 日本語
- 한국어
- Français
- Deutsch
- Español
- Italiano
- Русский

## 技术栈

- **前端框架**：Svelte 5 + SvelteKit
- **桌面框架**：Tauri 2
- **样式**：Tailwind CSS
- **Excel 处理**：xlsx
- **PDF 处理**：pdf-lib
- **二维码**：qrcode + jsqr
- **图片压缩**：browser-image-compression
- **文本对比**：diff
- **国际化**：svelte-i18n
- **语言**：TypeScript + Rust

## 安装说明

### macOS

如果打开应用时提示「已损坏」或「无法验证开发者」，请在终端执行：

```sh
xattr -cr /Applications/youran-toolbox.app
```

或者将 `/Applications/youran-toolbox.app` 替换为实际的应用路径。

### Windows

直接运行安装程序即可。

## 开发

### 安装依赖

```sh
bun install
```

### 开发运行

```sh
bun run tauri dev
```

### 构建应用

```sh
bun run tauri build
```
