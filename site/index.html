常见问题

macOS 提示「已损坏」或「无法验证开发者」怎么办？

在 macOS 上，从网络下载的应用会被系统打上 com.apple.quarantine 隔离属性。若应用未使用 Apple Developer ID 签名或尚未完成 Apple 公证，Gatekeeper 可能会拦截并提示「已损坏」或「无法验证开发者」。

建议先执行（仅移除隔离属性）：

$xattr -dr com.apple.quarantine "/Applications/YouranToolbox.app"

如果仍然无法打开，再执行（清空全部扩展属性）：

$xattr -cr "/Applications/YouranToolbox.app"

请将路径替换为你本机的实际应用路径。

Windows 安装常见问题

Windows 安装一般不会出现 macOS 的「已损坏」提示，但可能会遇到以下情况：

  • SmartScreen 提示「Windows 已保护你的电脑」：点击「更多信息」，再点击「仍要运行」。
  • 安装权限不足：建议右键安装程序，选择「以管理员身份运行」。
  • 安全软件拦截或误报：将安装程序加入白名单后重试。
  • 系统版本不符合：建议使用 Windows 10（1803+） 或 Windows 11 的 x64 系统。

平台说明

本应用基于 Tauri 构建，UI 层依赖操作系统内置的 WebView 组件渲染：macOS 使用 WebKit（Safari 内核），Windows 使用 WebView2（Edge 内核）。

  • macOS：仅支持 Apple Silicon 芯片（M1 及以上），WebKit 由系统提供，无需额外安装。
  • Windows：支持 Windows 10（1803+）及 Windows 11 的 64 位系统，WebView2 运行时已预装。
  • Linux：Tauri 依赖 WebKitGTK，各发行版的 WebKit 版本差异较大，暂不提供官方构建。

隐私政策

我们如何保护你的隐私

概述

悠然工具箱是一款完全本地运行的桌面应用，我们高度重视用户隐私。本应用不收集、不上传、不存储任何用户数据。

数据处理

  • 本地处理：所有文件和数据的处理均在你的设备上本地完成，不会传输到任何远程服务器。
  • 无网络请求：应用运行期间不会发起任何网络请求，完全离线可用。
  • 无数据收集：我们不收集任何使用数据、分析数据或个人信息。
  • 无追踪：应用内不包含任何追踪代码、广告 SDK 或第三方分析工具。

文件访问

  • 应用仅在你主动选择文件时访问对应文件，不会扫描或读取其他文件。
  • 处理结果仅保存到你指定的位置，不会创建隐藏副本。

开源透明

本应用完全开源，源代码托管在 GitHub，欢迎审查代码以验证我们的隐私承诺。

更新说明

本隐私政策可能随应用更新而调整，任何变更都会在此页面公布。
